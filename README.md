# Multi-AI Chat 🤖

A powerful chat interface that integrates multiple AI models for enhanced conversations, comparisons, and productivity.

## Features
- Support for multiple AI providers (OpenAI, Grok, etc.)
- Side-by-side model comparison
- Modern, responsive UI
- Conversation history & export

## Tech Stack
- Frontend: React / Next.js
- Backend: Python / Node.js
- AI Integration: LangChain or direct APIs

## Getting Started
1. Clone the repo
2. Install dependencies
3. Add your API keys
4. Run locally
